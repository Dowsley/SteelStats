# Gerenciador da Database
Dentro do JSON ```"config"``` estão as credenciais para o Banco de Dados que servirá como simulação dos dados da Gerdau para nosso MVP. *São informações confidenciais!*

O arquivo ```"manager.py"``` nos permite testar a conexão com o servidor, e também **inserir** no Banco de Dados todos os dados dos CSVs que Tiago passar. **AVISO: Não realize o procedimento de INSERÇÃO sem avisar ao grupo.**
