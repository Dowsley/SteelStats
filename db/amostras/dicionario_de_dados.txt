Dicionario de Dados

Cod_Equipment = código de identificação do equipamento
ID_Equipment_Status = id do evento de status do equipamento
COD_SHIFT = id do turno (manhã, tarde, noite)
COD_CREW = id da turma que está trabalhando
Equipment_Status_Mode = se o equipamento está ligado (1) ou desligado (0)
Unplanned_Time = tempo que ficou produzindo fora do programado
Stopped_Time = tempo que a maquina ficou parada
Production_Time = tempo que a maquina ficou produzindo
ID_Material_Definition = Id do material que estava sendo produzido
DT_START = inicio da marcação
DT_END = termino da marcação
QTY_Produced = total produzido
Metalic_Loss = perda metálica
Theoretical_Duration_By_PPH = quantidade de tempo que seria gasto para produzir o produto
THEORETICAL_DURATION_BY_PPH_FOR_CALCULATION = quantidade de tempo que seria gasto para produzir o produto que vai usar no calculo de OEE
